In the country of Ekkitilistan when recycling a bag of garbage, if the percentage of plastic in it is too low the whole bag is incinerated to save the effort of separating the plastic from the non-plastic items.

Certain cities in Ekkitilistan have different percentages, for example in Varaldreitilburg only $5\% $ may be non-plastic but in Ekkitilburg it is $7\% $.

You have decided to assist the recycling facilities in Ekkitilistan by writing a program that determines whether a bag of garbage satisfies the criterion.

Input

The first line of the input is the name of the city. This name doesn’t contain more than $10^5$ characters. The second line is the percentage $0 \leq p \leq 1$ of of the garbage bag that may be non-plastic. The number $p$ is a floating point number with precisely two digits after the decimal. The third line contains an integer $1 \leq n \leq 10^5$, the number of items in the garbage bag. The next $n$ lines describe the items in the bag. Each line is either the string “plast” indicating a plastic item or “ekki plast” indicating a non-plastic item.

Output

Print Jebb if the garbage bag is recyclable and print Neibb if it is not.
