Input

There is no input.

Output

An integer between $0$ and $1000$.

Explanation of Sample Inputs

In the sample the number $500$ is printed. A solution print ing the number $500$ gets $0$ points, because it is far from the answer.

Scoring

If $X$ is the distance between the correct answer and what your solution printed, then the solution gets $S = 100-X$ points. If $S$ is negative, then the solution gets $0$ points. If your solution gets $0$ points Kattis will give the verdict Wrong Answer.

500
